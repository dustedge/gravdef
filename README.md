## AGAB or a modernized clone of Gravity Defied
A practice project that is inspired by Gravity Defied and produced in a Godot Engine

### Current state: Playable

### Controls
- Arrow keys 	- Movement
- Enter 		- Restart level
- C 			- End race

- [X] Basic spritework
